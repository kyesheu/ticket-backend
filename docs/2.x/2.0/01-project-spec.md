# v2.0 项目规格

状态：已完成

## 范围

- 流程模板、节点、连线、发布版本和分类绑定。
- 串行人工节点、白名单条件分支和五种处理人规则。
- 完成、退回、取消、管理员终止及完整运行记录。
- v2.0 前历史工单继续使用原状态机。

## 规则

- workflow_key 与 version 唯一；草稿可改，发布后不可变。
- 新工单锁定当前发布版本；分类未绑定时使用内置标准流程。
- 条件仅允许优先级、分类和创建人部门，不接受脚本、SQL 或客户端表达式。
- 每个实例最多一个 PENDING 任务；并发完成最多一个请求成功。
- 任务、实例、工单状态、操作日志和通知在同一事务更新。

## 验收

流程图校验、版本锁定、三类条件、五种处理人、完成/退回/取消/终止、重复与并发提交、旧工单兼容及 v1.x 回归全部通过。
