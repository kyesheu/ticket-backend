# v2.1 项目规格

状态：已完成

## 范围

- 分类下管理文本、数字、日期、日期时间、单选、多选和布尔字段。
- 创建工单时校验字段并保存定义、值和选项标签快照。
- 详情按历史快照展示；流程条件支持 CUSTOM_FIELD 的 EQ 和 IN。

## 规则

- field_key 在分类内唯一，只允许大写字母、数字和下划线。
- field_key、field_type、category_id 创建后不可修改；定义只停用，不物理删除。
- 未知、重复、停用或跨分类字段一律拒绝。
- 工单、字段值和流程实例在同一事务保存。
- 自定义字段条件缺失或类型不兼容时不命中，走默认分支。

## 验收

七种类型、必填、长度、数值范围、选项、快照、停用字段、流程条件和注入防护全部正确；v1.0–v2.0 回归通过。
