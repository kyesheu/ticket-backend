# v2.3 项目规格

状态：已完成

## 范围

- Elasticsearch 检索工单编号、标题、描述和评论。
- 白名单过滤、安全高亮、稳定排序和 search_after 游标。
- 事务事件异步同步、失败重试与补偿。
- 版本化索引全量重建和别名原子切换。

## 规则

- MySQL 是唯一事实源，业务事务只写 ticket_search_event，不同步双写 ES。
- ES 先按服务端范围召回，ITicketAccessPolicy 再批量复核。
- 不接受客户端 DSL、脚本、索引名或字段名；页大小不超过 100。
- source_event_id 防止旧事件覆盖新快照。
- ES 故障不影响工单写入和普通 MySQL 列表。

## 验收

检索、组合过滤、权限、游标、高亮、事务一致性、重复/乱序消费、重试补偿、全量重建和故障降级全部通过。
