# v1.0 项目规格

状态：已完成

## 范围

- 工单创建、分页查询、详情、分派、处理、确认和取消。
- 树形分类、公开评论、内部备注和操作日志。
- 复用 Ticket 登录、用户、部门、角色和菜单权限。

## 状态机

NEW → PROCESSING → WAIT_CONFIRM → CLOSED；NEW 和 PROCESSING 可转为 CANCELLED。CLOSED、CANCELLED 为终态，每次流转必须写操作日志。

## 验收

1. 工单编号唯一，创建后状态为 NEW。
2. 分派、处理、确认、取消严格校验状态和操作者身份。
3. 详情返回分类、参与人、部门、评论和操作日志。
4. 列表支持分页及状态、优先级、分类、关键词和日期筛选。
5. 非法流转、越权操作和无效参数均被拒绝。
