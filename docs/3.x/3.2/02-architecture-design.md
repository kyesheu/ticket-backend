# v3.2 架构设计

状态：规划

## 知识管理

- Java 提供管理员入口和权限，不保存 Python 知识元数据副本。
- Python 提供文档列表、详情、删除和重建接口，Elasticsearch 保存元数据与有效代次。
- 删除以 sourceId 为边界，完成后刷新并验证不可检索；失败返回可重试状态。

## 反馈与指标

- Java 保存与当前用户、工单和建议记录关联的结构化反馈。
- 指标 Service 只读取结构化字段，不解析提示词或模型正文。
- Python 评测 runner 使用固定脱敏数据集和 fake/真实适配器，结果按模型、Embedding、索引和提示词版本区分。

## 安全

知识管理仅管理员可用；普通用户只能反馈自己有权访问工单的 AI 结果。删除、重新导入和评测操作写审计日志。
