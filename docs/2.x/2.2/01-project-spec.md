# v2.2 项目规格

状态：已完成

## 范围

- 工单正文和评论关联附件。
- 保存文件名、相对路径、MIME、扩展名、大小和上传人快照。
- 临时上传、受控绑定、查询、下载、逻辑删除和物理清理补偿。

## 规则

- 客户端不能提交服务器路径，响应不暴露绝对路径。
- 临时附件只能由上传人绑定一次，不得跨工单或业务类型绑定。
- 所有工单关联操作先执行对象级访问控制。
- 终态工单附件仅管理员可删除。
- 不做分片、秒传、预览、病毒扫描和对象存储。

## 验收

类型、空文件、大小和数量边界正确；绑定事务、归属、下载权限、路径安全、删除权限和清理补偿通过；v1.0–v2.1 回归通过。
