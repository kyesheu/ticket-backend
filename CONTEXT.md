# 工单与智能辅助 — 统一业务语言

**知识库** — 由导入文档和可复用历史工单组成的检索来源集合。

**知识文档** — 管理员导入知识库、可被切片和检索的独立资料。

**知识切片** — 从知识文档版本派生、用于检索并能反向定位原文的内容片段。

**相似历史工单** — 与当前工单语义相关、已形成有效处理结果的历史工单。

**工单辅助** — 根据当前工单和知识库生成处理建议与回复草稿的只读能力。

**引用** — 智能回答中指向可访问知识文档或工单事实的可核验依据。

**处理建议** — 根据工单事实和知识生成的、等待人工判断的非约束性草稿。

**回复草稿** — 工单辅助生成的、由处理人编辑后手工提交的回复文本。
