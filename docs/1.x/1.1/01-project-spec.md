# v1.1 项目规格

状态：已完成

## 范围

- 按 LOW、MEDIUM、HIGH、URGENT 配置 SLA 响应和解决时限。
- 创建工单时固化截止时间，列表和详情展示 SLA 状态。
- Quartz 扫描响应、解决超时并生成幂等告警。
- 管理员查询告警和手工补扫。

## 规则

- 使用 24×7 自然时间，不处理节假日和暂停计时。
- 策略修改不追溯已有工单；策略缺失或停用时拒绝创建对应优先级工单。
- 首次分派视为响应完成，确认关闭视为解决完成；CANCELLED 不再扫描。
- 同一工单、同一告警类型最多一条记录。

## 验收

四种策略唯一有效；截止时间快照正确；临界时间、重复扫描、并发扫描和终态判断正确；普通用户不能维护策略、查看全量告警或手工扫描。
