# v3.0 项目规格

状态：完成

## 范围

- 管理员导入 TXT、Markdown 和文本型 PDF 知识文档。
- 同步脱敏后的已关闭工单解决方案。
- 检索知识文档和相似历史工单。
- 生成带来源的处理建议和可编辑回复草稿。

## 规则

- Java 负责登录、对象权限、上下文和结果展示；Python 负责解析、切片、向量化、检索和生成。
- Python 不连接工单业务表，不接收用户 JWT，不修改工单。
- 仅 CLOSED 且有有效解决方案的工单可进入知识源。
- 无可靠证据、服务超时或模型不可用时明确降级。
- AI 输出不自动评论、分派或改变状态。
- 模型、Embedding 和向量库凭据只存在 Python 外部配置。

## 验收

文档导入、同步幂等、相似检索、来源校验、越权拦截、提示词注入防护和故障降级通过；Java、Python、Elasticsearch 完整端到端门禁通过后才能标记完成。
